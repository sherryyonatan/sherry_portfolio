עמוד הצגת הפרויקט

ניהול משימות יום-יומי קסום, הכולל בחירת תאריך, נושא יומי, רשימת משימות לפי שעות, חישוב אחוזי התקדמות בזמן אמת, שמירה ב-LocalStorage, מחיקת רשימה, והעברת משימות למחר באופן אוטומטי.

טכנולוגיות הפרויקט

HTML CSS JavaScript LocalStorage DOM
לצפייה בפרויקט להורדת תיקיית ZIP של הפרויקט

מה כולל הפרויקט?

  • בחירת תאריך ונושא יומי
  • רשימת משימות לפי שעות
  • סימון משימות שבוצעו
  • חישוב אחוזי התקדמות בזמן אמת
  • שמירה אוטומטית ב-LocalStorage
  • מחיקת רשימה
  • העברת משימות למחר

מטרת הפרויקט

מטרת הפרויקט הייתה לתרגל עבודה עם DOM, יצירת רכיבים דינמית, localStorage, עבודה עם תאריכים, חישוב התקדמות, אפיון ממשק נוח וברור לניהול משימות יומיומי.
