עמוד הצגת הפרויקט

פרויקט פאקמן הכולל לוח משחק דינמי, מערכת חיים, ניקוד, Power Mode, תנועת אויבים, אנימציות וסאונדים. המשחק נבנה באמצעות JavaScript עם לוגיקה מורכבת לניהול מצבים.

טכנולוגיות הפרויקט

HTML CSS JavaScript Game Logic LocalStorage
לצפייה בפרויקט להורדת תיקיית ZIP של הפרויקט

מה כולל הפרויקט?

  • מסך פתיחה עם הזנת שם שחקן
  • לוח משחק דינמי מבוסס grid
  • מערכת חיים וניקוד
  • Power Mode עם טיימר
  • תנועת Ghosts על הלוח
  • מסכי ניצחון והפסד
  • שיא אישי ב־localStorage

מטרת הפרויקט

מטרת הפרויקט הייתה לתרגל לוגיקה של משחק בזמן אמת, עבודה עם DOM, timers, state management, localStorage ואנימציות, תוך בניית ממשק משחק עשיר, אינטראקטיבי ומהנה למשתמש.
