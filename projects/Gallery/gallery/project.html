עמוד הצגת הפרויקט

מערכת גלריית תמונות מתקדמת המדמה חוויית משתמש בסגנון רשת חברתית. הכוללת אלבומים, העלאת תמונות, לייקים, תגובות, צפייה במסך מלא יצירת משתמשים לניהול וצפייה באלבום ושמירת נתונים ב־LocalStorage.

טכנולוגיות הפרויקט

HTML CSS JavaScript LocalStorage JSON

מה כולל הפרויקט?

  • ניהול אלבומים ותמונות
  • הקמת משתמשים עם הרשאות
  • מערכת לייקים ותגובות
  • הוספה ומחיקה של תמונות
  • שמירת תמונה במחשב

מטרת הפרויקט

לתרגל עבודה עם מבני נתונים מורכבים, DOM דינמי, אינטראקציות משתמש, וליצור חוויית משתמש עשירה המדמה אפליקציית גלריה אמיתית.
