עמוד הצגת הפרויקט

משחק זיכרון אינטראקטיבי עד ארבעה משתתפים או למשחק מול המחשב. המשחק כולל מסך הזנת שמות, בחירת נושא הקלפים, טיימר לכל תור, סאונדים, ניקוד דינמי ומסך ניצחון יחודי לכל נושא קלפים. הפרויקט נבנה מתוך חשיבה על חוויית משתמש ברורה, משחקיות זורמת ולוגיקה דינמית ב-JavaScript.

טכנולוגיות הפרויקט

HTML CSS JavaScript Game Logic Responsive UI

מה כולל הפרויקט?

  • בחירת מצב משחק: מול מחשב או עד ארבעה שחקנים
  • הזנת שמות השחקנים
  • בחירת נושא למשחק
  • טיימר לכל תור
  • ניקוד דינמי לכל שחקן
  • לוגיקה של התאמת זוגות
  • מסך ניצחון וקונפטי לפי נושא

מטרת הפרויקט

מטרת הפרויקט הייתה לתרגל עבודה עם DOM, state management, אירועים, טיימר, לוגיקה של משחק רב-שלבי, והצגת מידע דינמית במסכים שונים תוך יצירת חוויית משתמש ידידותית ומעוצבת.
