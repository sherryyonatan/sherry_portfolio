הפרויקטים בפורטפוליו:
1. משחק X-O (איקס עיגול)
משחק לשני שחקנים עם מערכת ניקוד, טיימר ותצוגה דינמית.
יכולות מרכזיות:
ניהול תורות בין שחקנים
זיהוי ניצחון / תיקו
טיימר לכל תור
פופאפ ניצחון מעוצב + אנימציות
שמירת ניקוד בזמן אמת

 2. משחק זיכרון (Memory Game)
משחק קלפים אינטראקטיבי עם אפשרות לעד 4 שחקנים.
יכולות מרכזיות:
בחירת נושא (פרחים, חלל, קינוחים וכו׳)
תמיכה במצב שחקן מול מחשב
טיימר לכל תור
מערכת ניקוד לפי התאמות
אפקטים קוליים ואנימציות (כולל קונפטי)

 3. משחק Pac-Man
משחק קלאסי בגרסה מודרנית עם חוויית משתמש עשירה.
יכולות מרכזיות:
שליטה בדמות באמצעות מקלדת
מערכת חיים (3 לבבות)
טיימר + שיאים (LocalStorage)
Power Mode (אכילת רוחות)
מסכי ניצחון / הפסד עם אנימציות וסאונד

 4. אפליקציית To-Do List
ניהול משימות יומי עם מעקב התקדמות.
יכולות מרכזיות:
יצירת משימות לפי תאריך ונושא
חישוב אחוז השלמה אוטומטי
שמירה אוטומטית (LocalStorage)
מעבר בין ימים ושמירת היסטוריה
ממשק נקי ונוח לשימוש

 5. אפליקציית מזג אוויר
אפליקציה דינמית המציגה נתוני מזג אוויר בזמן אמת.
יכולות מרכזיות:
חיפוש ערים
שימוש ב-API חיצוני
תחזית לפי שעות ולפי ימים
שינוי רקע לפי יום/לילה
הצגת מדדים: טמפרטורה, רוח, לחות, ראות ועוד

 6. גלריית תמונות אינטראקטיבית
מערכת ניהול אלבומים ותמונות עם הרשאות משתמשים.
יכולות מרכזיות:
יצירת אלבומים דינמית
הוספת תמונות
לייקים ותגובות
הורדת תמונות
מחיקת תוכן (לפי הרשאות)
מערכת משתמשים בסיסית

 מטרות הפרויקטים
יישום ידע תיאורטי בפרויקטים אמיתיים
